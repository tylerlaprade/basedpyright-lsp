# basedpyright-lsp

A Claude Code LSP plugin for Python using [basedpyright](https://github.com/DetachHead/basedpyright).

## Prerequisites

Install basedpyright:

```bash
pip install basedpyright
# or
uv pip install basedpyright
```

## Installation

```bash
claude plugin add tylerlaprade/basedpyright-lsp
```

## Why basedpyright?

[basedpyright](https://github.com/DetachHead/basedpyright) is a fork of pyright with:
- Additional type checking rules
- Stricter defaults
- Better error messages
- Active community development
